## ✨ Portfolio Features

| Feature | Description |
|---|---|
| 🎨 **Dark Glassmorphism UI** | Modern dark theme with Electric Blue accents |
| ✍️ **Typing Effect** | Animated role text (Backend Architect / Full-Stack Engineer / Blockchain Enthusiast) |
| 🌐 **Particle Network BG** | Interactive 3D node network that reacts to mouse movement |
| 🖱️ **Custom Glow Cursor** | Glowing Electric Blue cursor with a smooth trailing ring |
| 🃏 **3D Card Tilt** | Project cards and skill tiles tilt in 3D on hover (via VanillaTilt.js) |
| 📊 **Scroll Progress Bar** | Thin glowing line fills as you scroll |
| 🚀 **Back-to-Top Button** | Appears after scrolling and returns to top smoothly |
| 📧 **Contact Form** | Live email delivery via EmailJS with real-time validation |
| 📱 **Fully Responsive** | Optimised for mobile, tablet, and desktop |

## 🏃 Running Locally

```bash
# Install serve (Node.js required)
npm install -g serve

# Navigate to the project folder
cd myportfolio

# Start the local server
npx serve -p 3000
```

Open your browser and visit: **http://localhost:3000**
